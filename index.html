About Lyricsology:

Description:

Find song lyrics easily with Lyricsology. Search by title and artist, explore detailed track information, view artist images, and get a preview of lyrics.

Technologies Used:

JavaScript ● HTML ● Css/Sass
Musixmatch API ● Genius API

Features:

  • Search Lyrics:
    Find song lyrics by entering the title and artist.

  • Detailed Information:
    Explore detailed information about a track, including artist images and song art.

  • Lyrics Preview:
    Get a preview of lyrics for the searched track.

  • Bookmark (coming soon):
    Bookmark your favorite musics
